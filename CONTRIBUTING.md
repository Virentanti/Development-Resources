# Contribution

Contribute as many development resources that you'd like. Please make sure contributions are valid resources and ideally are free for other developers. Your contributions to this project are appreciated!

## Contributing Resources

In order to add a new development resource, you need to follow these steps:

- Fork the [main repo](https://github.com/nataliepina/Development-Resources.git).
- Edit the README.md to include your new resources and stay consistent with the format of the other resources.
- Create a new Pull Request when you're ready to submit your changes.
- Now, just wait for your PR to be approved and merged!
